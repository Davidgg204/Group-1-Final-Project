# Group 1 Final Project Weather App

A full-stack weather web application built using React, Node.js, and Open-Meteo API.

## Features
- Search weather by city
- 5-day forecast
- Favorites system
- Search history
- Responsive UI

## Link to the project
https://group-1-final-project.onrender.com

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- Open-Meteo API
- Render (Deployment)
